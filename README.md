## üß™ Challenges Faced & Solutions Implemented

### **1. Understanding Next.js Routing**
- **Challenge:** Confusion around App Router vs. Pages Router.
- **Solution:** Practiced building small Next.js apps and using file-based routing to reinforce concepts.

### **2. Styling Consistency with TailwindCSS**
- **Challenge:** Maintaining consistent design across pages.
- **Solution:** Implemented shared component styles and Tailwind config customization.

### **3. API Integration Errors**
- **Challenge:** Handling asynchronous data and error states.
- **Solution:** Adopted try/catch patterns, loading states, and reusable API helpers.

### **4. TypeScript Strict Mode**
- **Challenge:** Frequent type errors.
- **Solution:** Learned proper type definitions, interfaces, and union types.

### **5. GraphQL Query Structure**
- **Challenge:** Understanding queries, mutations, and resolvers.
- **Solution:** Practiced schema design and tested queries using Playground/Insomnia.
